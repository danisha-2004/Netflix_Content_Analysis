# Netflix Content Analysis

## Project Overview

This project focuses on analyzing Netflix content trends, distribution patterns, and platform growth using Exploratory Data Analysis (EDA). The objective is to understand how Netflix content varies across countries, genres, ratings, release periods, and content types while generating meaningful business insights.

## Objectives

* Analyze Netflix content distribution patterns
* Identify trends in content growth over time
* Understand content contribution across countries
* Explore ratings, genres, and duration patterns
* Generate business insights from Netflix content data

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)

6. Trend Analysis
7. Insight Generation

## Analysis Performed

* Movies vs TV Shows Analysis
* Content Growth Analysis
* Monthly Content Addition Analysis
* Country-wise Content Distribution
* Ratings Distribution Analysis
* Genre Analysis
* Movie Duration Analysis
* Director Analysis
* Cast Analysis
* Trend and Relationship Analysis

## Key Insights

* Netflix content catalog is dominated by movies compared to TV shows.

* Content additions increased rapidly after 2015, indicating significant platform expansion.

* The United States contributes the highest amount of Netflix content.

* International content production has increased significantly over time.

* TV-MA and TV-14 ratings dominate the platform.

* Most movie durations fall within the medium-duration category.

* Netflix focuses heavily on movie-based content distribution and global expansion strategies.

## Conclusion

The analysis suggests that Netflix has adopted a strategy focused on rapid content expansion, movie-dominated distribution, international growth, and diversified content offerings. Content trends indicate strong global expansion and increasing focus on mature audience segments.
